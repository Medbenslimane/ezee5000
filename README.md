# 🛡️ AdBlocker Shield

برنامج سطح مكتب احترافي لإزالة الإعلانات المزعجة من هواتف Android بدون فقدان البيانات أو المساس بملفات النظام.

---

## ✨ المميزات

- 🚫 **حجب الإعلانات عبر DNS** — يستخدم خوادم DNS متخصصة (AdGuard, NextDNS, Cloudflare, Quad9)
- 📱 **اتصال آمن عبر ADB** — يتصل بالهاتف عبر USB بدون الحاجة لصلاحيات Root
- 🔒 **آمن تماماً** — لا يحذف بيانات ولا يعدل ملفات النظام
- 📋 **قوائم حجب محدثة** — يدعم مصادر متعددة مع تحديث تلقائي
- 📊 **إحصائيات مفصلة** — يتتبع عدد النطاقات المحجوبة والجلسات
- 🎨 **واجهة حديثة** — واجهة رسومية أنيقة تدعم الوضع الداكن والفاتح
- ⚙️ **قابل للتخصيص** — إضافة نطاقات مخصصة وقائمة بيضاء

---

## 📋 المتطلبات

### النظام
- Windows 10/11 أو macOS أو Linux
- Python 3.8 أو أحدث
- Android Platform Tools (ADB)

### الهاتف
- Android 9 أو أحدث (API 28+)
- تفعيل وضع المطور (Developer Mode)
- تفعيل تصحيح USB (USB Debugging)

## 🚀 التثبيت

### 1. تثبيت Python
قم بتحميل Python من [python.org](https://www.python.org/downloads/)

### 2. تثبيت ADB (Android Platform Tools)
- **Windows**: حمّل من [developer.android.com](https://developer.android.com/tools/releases/platform-tools)
- **macOS**: `brew install android-platform-tools`
- **Linux**: `sudo apt install adb`

### 3. تثبيت البرنامج

```bash
# استنساخ المشروع
git clone https://github.com/Medbenslimane/ezee5000.git
cd ezee5000/AdBlocker

# تثبيت المكتبات المطلوبة
pip install -r requirements.txt

# تشغيل البرنامج
python main.py
```

## 📱 كيفية الاستخدام

### إعداد الهاتف
1. افتح **إعدادات الهاتف** > **حول الهاتف**
2. انقر على **رقم الإصدار** 7 مرات لتفعيل وضع المطور
3. ارجع إلى **الإعدادات** > **خيارات المطور**
4. فعّل **تصحيح USB**
5. وصّل الهاتف بالكمبيوتر عبر كابل USB
6. اقبل رسالة الإذن التي تظهر على الهاتف

### استخدام البرنامج
1. شغّل البرنامج: `python main.py`
2. اضغط على **📱 الجهاز** > **🔌 اتصال بالجهاز**
3. اختر خادم DNS من صفحة **🚫 حجب الإعلانات**
4. اضغط **✅ تفعيل الحماية**

## 🔧 كيف يعمل البرنامج

البرنامج يستخدم أسلوباً **آمناً بالكامل** لحجب الإعلانات:

1. **Private DNS**: يُغيّر إعدادات DNS الخاصة على الهاتف إلى خادم DNS يحجب الإعلانات (مثل AdGuard DNS)
2. **بدون Root**: جميع العمليات تتم عبر ADB بصلاحيات المستخدم العادي
3. **قابل للتراجع**: يمكن إعادة كل شيء للوضع الأصلي بضغطة زر
4. **لا يمس ملفات النظام**: لا يعدل أي ملف في قسم `/system`

---

## ⚠️ ملاحظات هامة

- البرنامج **لا يحتاج صلاحيات Root** على الهاتف
- **لا يتم حذف** أي تطبيق أو بيانات من الهاتف
- تعطيل التطبيقات الإعلانية **قابل للتراجع** بالكامل
- يُنصح بعمل نسخة احتياطية قبل أي تغييرات (كإجراء احتياطي)
- يتطلب أن يظل الهاتف متصلاً عبر USB أثناء التكوين فقط
