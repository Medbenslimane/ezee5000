# 🌐 Ezee Browser

متصفح إنترنت احترافي مبني على Electron بميزات متقدمة ومتميزة عن المتصفحات التقليدية.

## ✨ الميزات الرئيسية

### 🛡️ الخصوصية والأمان
- **حظر الإعلانات المدمج** - يحظر 20+ شبكة إعلانية تلقائياً
- **حظر المتتبعات** - يمنع أكثر من 10 شبكات تتبع
- **مسح البيانات** - حذف الكوكيز وبيانات التصفح بنقرة واحدة
- **مؤشر الأمان** - يظهر حالة SSL للمواقع

### 🎨 الواجهة
- **الوضع المظلم/الفاتح** - تبديل تلقائي حسب وقت اليوم أو يدوي
- **تبويبات متعددة** - إدارة كاملة مع إغلاق وإضافة
- **تصميم عصري** - واجهة أنيقة ومتجاوبة
- **اختصارات لوحة المفاتيح** - تصفح سريع بدون ماوس

### 🧰 أدوات مدمجة
- **📸 لقطة شاشة** - التقاط الصفحة وحفظها كصورة
- **📥 مدير التنزيلات** - مع شريط التقدم
- **📝 ملاحظات جانبية** - دوّن أفكارك أثناء التصفح
- **⏱️ مؤقت** - عداد زمني مدمج
- **🧮 آلة حاسبة** - حسابات سريعة
- **🌐 ترجمة** - ترجمة فورية
- **📱 QR Code** - إنشاء رمز QR لأي صفحة
- **🎨 منتقي الألوان** - اختيار ألوان للمصممين

### 📋 إدارة التصفح
- **المفضلة** - حفظ المواقع المفضلة
- **السجل** - تتبع المواقع المزارة
- **صفحة رئيسية ذكية** - اختصارات سريعة وإحصائيات

## 🚀 التشغيل

```bash
# تثبيت التبعيات
npm install

# تشغيل المتصفح
npm start
```

## ⌨️ اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl + T` | تبويب جديد |
| `Ctrl + W` | إغلاق التبويب |
| `Ctrl + L` | التركيز على شريط العناوين |
| `Ctrl + R` | تحديث الصفحة |
| `Ctrl + D` | إضافة/إزالة مفضلة |
| `Ctrl + B` | الشريط الجانبي |
| `F11` | ملء الشاشة |
| `F12` | أدوات المطور |

## 🛠️ التقنيات المستخدمة

- **Electron** - إطار العمل الرئيسي
- **HTML5/CSS3** - الواجهة
- **JavaScript** - المنطق البرمجي
- **Font Awesome** - الأيقونات
